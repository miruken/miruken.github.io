======
Sphinx
======

Versions
========

* Sphinx 1.5.6
* SphinxContrib-Versioning 2.2.1 

Specifically using these versions because of compatibility issues with :code:`SphinxContrib-Versioning`
and Sphinx 1.6.5

.. code-block:: none

	pip install sphinx==1.5.6
	pip install sphinxcontrib-versioning==2.2.1
